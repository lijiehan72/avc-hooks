# avc-hooks

## Install

```bash
npm install --save avc-hooks
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'avc-hooks'
import 'avc-hooks/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
